Quantum Computing

Quantum computing has been a topic of discussion for a good while now, but what even is quantum computing, and why is it important? The computers we know today store information using bits that are either 0 or 1. Quantum computing takes this same idea and upgrades it using something called qubits. These qubits are different, because they can be both 0 and 1 at the same time and linking qubits to each other to solve problems using quantum physics. This means that quantum computers will be able solve problems faster than today's computing and even handle problems that today's computing can't.
